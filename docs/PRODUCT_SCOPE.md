# pdf-namefix Product Scope

## Product Summary

`pdf-namefix` is a local-first CLI tool that helps users preview and safely rename messy PDF files.

It is designed for people who collect many PDFs such as:

- invoices
- receipts
- contracts
- reports
- statements
- tax documents
- offers
- applications
- books
- ebooks
- course materials
- lesson notes
- papers
- articles
- slides
- manuals
- transcripts
- worksheets
- guides
- cheatsheets

The tool should be useful for both personal file organization and developer-style automation workflows.

## Core Problem

PDF files often arrive with unclear names:

```text
scan.pdf
document.pdf
invoice-final.pdf
whatsapp-doc.pdf
lecture1.pdf
book-new.pdf
