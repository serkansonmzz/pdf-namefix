# pdf-namefix Roadmap

## Phase 0 = Product Definition and Scope

Define the product, v0.1 scope, safety rules, document types, and future boundaries.

## Phase 1 = Project Skeleton

Set up the Python package, CLI entry point, test framework, and basic project structure.

## Phase 2 = Multi-folder PDF Scanner

Scan one or more folders and find PDF files safely.

## Phase 3 = Document Type Classifier

Classify PDFs using filename and simple local text/metadata rules.

## Phase 4 = Filename Suggestion Engine

Generate safe, readable filename suggestions.

## Phase 5 = Preview Command

Show proposed renames without touching files.

## Phase 6 = Safe Apply / Rename Flow

Apply renames only when explicitly requested and record rename history.

## Phase 7 = Downloads Cleaner Layer

Organize PDFs into type-based folders.

## Phase 8 = Tests and Hardening

Add edge-case-driven tests and improve reliability.

## Phase 9 = README and Demo

Document installation, usage, examples, and safety behavior.

## Phase 10 = v0.1 Release Candidate

Prepare the first GitHub release candidate.
